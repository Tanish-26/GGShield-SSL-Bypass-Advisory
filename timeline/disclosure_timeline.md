# Disclosure Timeline — GGShield SSL/TLS Verification Bypass


- **2025-09-20**: Initial vulnerability report submitted to GitGuardian.
- **2025-10-13**: Vendor responded, termed the behavior an "intentional design choice".
- **2025-10-13**: Escalated to GitHub Security Lab.
- **2025-10-15**: GitHub Security Lab dismissed the escalation.
- **2025-10-20**: Vendor refused substantive remediation — offered documentation changes only.
- **2025-11-13**: CERT/CC reviewed and recommended public disclosure due to vendor non-cooperation.
- **2025-11-17**: Final notice sent to vendor offering a last opportunity to coordinate before disclosure.
- **2025-11-20**: Public disclosure planned (updated: CVE cancelled — advisory published independently).
